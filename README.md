# directorier
Creates a directory tree.

## Installation

```
$ npm install directorier --save
```

## Usage

```
$ directorier output [flag]
```

### Flag

| Flag | Short Flag | Description |
|---|---|---|
| --help | -h | output usage information |
| -- path | -p | Specify directory path |

### Result

```
directorier
├─ .gitignore
├─ .npmignore
├─ index.js
├─ lib
│  ├─ customHelp.js
│  ├─ directoryTree.js
│  ├─ index.js
│  └─ version.js
├─ LICENSE
├─ package-lock.json
├─ package.json
└─ README.md

Directories: 2, files: 11
```
